# Asura: minimalist social app.

## A minimalist news feed app built with Angular.

![That's the home screen](./public/assets/images/screenshot-01.png)

## Ingredients
- Angular 18.0
- Bootstrap 5.3
- Bootstrap icons

## Build the project

Just the two commands `npm i` and `npm start`.
